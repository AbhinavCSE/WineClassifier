# WineClassifier
Classifying Wine based on the given attribute values

#The Data
We have been given a dataset about wines with the following attributes
  1.ID
  2.Alcohol
  3.Malic Acid
  4.Ash
  5.Alcalinity of ash
  6.Magnesium
  7.Total pheonls
  8.Flavanoids
  9.Nonflavanoid phenols
 10.Proanthocyanins
 11.Color intensity
 12.Hue
 13.OD280/OD315 of diluted wines
 14.Proline
